## CARD ARCHITECTURE (3 panels)
1. **Listing View** — Photo gallery, address, beds/baths/sqft, tier badge, metal nameplate, rarity label
2. **Value View** — Current Value → Purchase Price → Equity → Mortgage Balance → Delta → % Increase → DSCR
3. **P&L View** — Revenue rows, expense rows, NOI, improvements log, forecasts

### Navigation
- Property name stays on every panel (nav bar top)
- Dot indicators in nav bar
- translateX slide animation (no rotateY — causes twerk)
- Panel 0 = listing, Panel 1 = value, Panel 2 = P&L
